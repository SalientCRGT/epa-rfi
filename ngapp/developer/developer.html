H-API-Gen

H-API-Gen takes existing datasets and Hapi-ly wraps them in a generated JSON API ready to be consumed by you, one very Hapi developer! Simply search, get results, and apply them to your own application. For now, we have chosen the Facilities Registry System dataset in csv format, but we will Hapi-ly expand our application to offer other datasets and file formats as well! HapiGen-erating!

Sample Queries

Everything /api/facilities
Returns all facilities.

Attribute Filtering /api/facilities?cityName=chantilly
Returns all facilities with a city name containing "chantilly".

Regular Expressions /api/facilities?postalCode=20151|20171|22033
Returns all facilities with a postal code containing 20151, 20171 or 22033.
